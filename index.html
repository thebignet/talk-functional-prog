Fonction pure

	return x++;
}

incr n'est pas une fonction pure

Toutes les propriétés et bénéfices de la programmation fonctionnelle sont dérivées de l'absence d'effets secondaires lors de l'application d'une fonction

Fonctions d'ordre supérieur

Fonctions qui prennent d'autres fonctions en paramètre

mult(x,y) → x * y

paire(f,x) → f(x,x)

carre(x) → paire(mult,x)

Fonctions qui retourne une fonction

puissance(n) → x → x ^ n

Exemple

Fonction en paramètre

def mult(a: Long, b: Long): Long = a * b
def paire(f: (Long, Long) => Long, x: Long) = f(x, x)
def carre(x: Long) = paire(mult, x)

Exemple

Fonction en retour

def puissance(x:Long) = (y:Long) => Math.pow(y,x)
def carre2(x: Long) = puissance(2)(x)

Pattern matching

def titre(x: Any): String = x match {
  case 1 => "un"
  case "Jean" => "Juan"
  case Personne(nom, age) => s"Monsieur ${nom} agé de ${age} ans"
  case nom: String => nom
  case _ => ""
}
